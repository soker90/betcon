Después de casi dos años sin actualizaciones públicas, llega la primera versión 2.x de Betcon. Esta versión trae una renovación importante de la interfaz y mejoras en el día a día de uso.

Novedades principales:

```
1. Nueva interfaz más moderna y fluida
2. Modo oscuro
3. Mayor rendimiento general de la aplicación
4. Barra de progreso al importar archivos CSV
5. Nuevos idiomas: francés, italiano y portugués de Portugal
6. Traducciones actualizadas en todos los idiomas disponibles
7. Disponible para Windows, macOS y GNU/Linux
```

La aplicación está disponible en español, inglés, alemán, francés, italiano, portugués, turco y kurdo.

**GNU/Linux**

Arch Linux — desde AUR:
```bash
yay -S betcon
```

Ubuntu/Debian — desde el PPA oficial:
```bash
sudo add-apt-repository ppa:soker/betcon
sudo apt update
sudo apt install betcon
```
