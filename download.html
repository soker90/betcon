GNU/Linux


ArchLinux

~$ yaourt -Sy betcon

Ubuntu

~# add-apt-repository ppa:soker/betcon
~# apt-get update
~# sudo apt-get install betcon

Debian

~# apt-key adv --keyserver keyserver.ubuntu.com --recv-keys FA4746D8
~# echo 'deb http://ppa.launchpad.net/soker/betcon/ubuntu xenial main #Betcon' >> /etc/apt/sources.list
~# apt-get update
~# sudo apt-get install betcon

Desde el código fuente

Antes de instalar, asegúrate de tener instaladas las dependencias:

  • Python3
  • PyQt5

Una vez satisfechas, instala con:

~# make install
